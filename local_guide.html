NABIO STORE

SISTEMA DINÂMICO COM BANCO DE DADOS

Você abriu o projeto usando o **Go Live (Live Server)**. Como este sistema possui um servidor backend seguro com banco de dados SQLite, ele não roda como uma página estática comum. Siga as instruções abaixo para ligá-lo localmente:

1

Abra o Terminal

No VS Code, abra o terminal integrado pressionando Ctrl + ` (ou vá em Terminal > Novo Terminal).

2

Inicie o Servidor Node.js

Digite o comando abaixo no terminal e aperte Enter:
node server.js

3

Acesse no Navegador

Com o servidor ativo, abra o navegador e acesse o endereço oficial:

Acessar Nabio Store (http://localhost:3000)
☁️ Hospedagem na Hostinger:
Quando você subir para a Hostinger, configure uma nova Node.js Application, aponte a pasta raiz e defina server.js como o arquivo de inicialização (Application Startup File). A Hostinger executará o comando automaticamente e fará o mapeamento do seu domínio!
